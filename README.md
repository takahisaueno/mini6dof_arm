# mini6dof_arm_project

このプロジェクトは小型で高機能な6軸のロボットアームのプラットフォームをROSで構築することを目標にした**個人の趣味プロジェクト**である。
このプロジェクトでは以下のタスクを達成することを目標にしている
- 電流監視ベースの外力推定
- 電流制御ベースのインピーダンス制御
- yoloなどの画像認識モジュールを用いた開発
- 深層予測学習ベースの物体のピッキングタスクの実現
